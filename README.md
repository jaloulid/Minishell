# Minishell
Minishell — Custom Unix Shell in C
Minishell is a fully functional Unix shell developed in C, built from the ground up to replicate core behaviors of bash. This project demonstrates a deep understanding of operating systems concepts, including process creation, file descriptors, signals, and inter-process communication. It supports command execution, pipes, redirections, environment variable expansion, and built-in commands, all handled through a custom parsing and execution pipeline.

A key challenge was designing a robust parser capable of handling complex command structures while maintaining correctness and stability. I implemented precise tokenization, syntax validation, and an execution engine that efficiently manages processes and resources. Special attention was given to signal handling and memory management to ensure a smooth and reliable user experience.

Minishell reflects my ability to build complex, low-level systems, debug intricate behaviors, and deliver clean, maintainable, and efficient C code in a Unix environment.
